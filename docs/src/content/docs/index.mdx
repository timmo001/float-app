`float-app` identifies Hyprland windows and writes persistent floating rules for their exact class or initial class.

<CardGrid>
  <Card title="Pick a window" icon="cursor">
    Select any visible window with `slurp`, without changing keyboard focus.
  </Card>
  <Card title="Persistent rules" icon="document">
    Keep managed rules in a generated file and include it from your chosen
    Hyprland config.
  </Card>
  <Card title="Config-safe writes" icon="setting">
    Preserve stowed symlinks, support Lua and conf configs, and avoid replacing
    Omarchy's shared floating rules.
  </Card>
</CardGrid>
