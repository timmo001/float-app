# float-app Docs

The [float-app](https://github.com/timmo001/float-app) documentation site, built with Astro and Starlight.

Run these commands from the repository root:

- `mise run docs:install`
- `mise run docs:gen:cli`
- `mise run docs:build`

From this directory, use `bun run dev` for local development and `bun run deploy` to deploy the static site to Cloudflare Workers.
